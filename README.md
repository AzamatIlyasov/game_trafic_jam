# 🚦 Симулятор светофоров и пробок

Браузерная игра-симулятор дорожного движения для телефона (iPhone Safari) и десктопа.
Главная фишка — **«фантомная» пробка**: затор, возникающий «из ниоткуда», без аварии
и без узкого места, чисто из динамики следования машин друг за другом.

## Возможности

- ⭕ **Кольцо** — классический эксперимент Сугиямы: подними плотность, нажми 💥 —
  и волна затора побежит назад по потоку.
- 🛣 **Магистраль** — настраиваемое число перекрёстков (1–12), дистанция между ними
  (60–500 м), интенсивность потока на въезде.
- 🚦 **Светофоры** — длительность зелёного/красного, «зелёная волна» (сдвиг фаз),
  🧠 адаптивный режим (светофор смотрит на очереди).
- 📈 **Диаграмма пространство-время** — фантомная пробка видна как красные
  диагональные полосы.
- Метрики: число машин, средняя скорость, поток, доля стоящих, суммарное ожидание.
- Пресеты: Кольцо Сугиямы, Час пик, Зелёная волна, Умные светофоры.

## Физика

Модель [Intelligent Driver Model (IDM)](https://en.wikipedia.org/wiki/Intelligent_driver_model):
каждая машина непрерывно выбирает ускорение по дистанции и скорости сближения с лидером.
Красный светофор — «виртуальная неподвижная машина» на стоп-линии. Выше критической
плотности модель строчно-неустойчива — малые возмущения растут в устойчивые stop-and-go
волны, бегущие назад со скоростью ~15–20 км/ч, как в реальности.

## Разработка

```bash
npm install
npm run dev        # локальный сервер
npm run build      # сборка в dist/
npm run sim:check  # headless-проверки физики (без браузера)
```

## Деплой

Пуш в `main` автоматически собирает и публикует игру в GitHub Pages
(`.github/workflows/deploy.yml`). В настройках репозитория один раз включите
**Settings → Pages → Source: GitHub Actions**.

Подробный план разработки: [PLAN.md](PLAN.md).
